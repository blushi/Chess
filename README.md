# Getting Started

## Requirements

* Python 3.4+

## Installation

Chess can be installed with pip:

```
$ pip install Chess
```

or directly from the source code:

```
$ git clone https://github.com/theovoss/chess.git
$ cd chess
$ python setup.py install
```

# Basic Usage

After installation, the package can imported:

```
$ python
>>> import chess
>>> chess.__version__
```
